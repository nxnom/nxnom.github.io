My Coding Journey in a Nutshell

I started exploring programming back in 2011 during my first year at university. I was into coding for three years but then took a break for five years, mainly using tools like WordPress. In 2019, I decided to code again because I wanted to turn my WordPress website into an Android app.

I couldn't find the right open-source project, so I had no choice but to get back into coding. Flutter was popular at that time, so I began learning it, and I've been using it ever since.

When I shared my projects on social media, people started asking me to create apps for them. That's when I started freelancing. I realized that I needed to learn JavaScript, React.js for web development, and Node.js for backend development to expand my skill set.

I'm passionate about learning and staying updated with industry trends. Through my projects, I've developed problem-solving, communication, and teamwork skills. I'm excited about the future and eager to work on exciting projects.
